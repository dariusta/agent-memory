# Tag Taxonomy

Use canonical tags from this list when creating or updating pages. Add new canonical tags here rather than inventing one-off variants.

## Visibility (optional, controls wiki-query filtered mode)

- `visibility/public` — safe to surface anywhere (default if unset)
- `visibility/internal` — exclude from "public only" / "user-facing" queries
- `visibility/pii` — contains personal data; excluded from filtered queries
